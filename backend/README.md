# Backend (Node + Express)

## Setup

```bash
cd backend
npm install
npm run dev
```

## Environment variables

- `PORT` (default: `4000`)
- `CORS_ORIGIN` (single origin or comma-separated list)
- `DB_HOST`
- `DB_PORT`
- `DB_NAME`
- `DB_USER`
- `DB_PASSWORD`

## Endpoints

- `GET /api`
- `GET /api/health`
